# SocialMock Pro 🚀

SocialMock is a high-fidelity WhatsApp chat generator designed for UI/UX designers, marketers, and educators. It allows users to create pixel-perfect, high-resolution chat mockups with full customization.

## ✨ Features
- **3X Resolution Export:** High-quality PNGs without watermarks.
- **Full UI Control:** Customize names, profile pics, and timestamps.
- **Privacy Toggles:** Add or remove end-to-end encryption notices.
- **Chrome Optimized:** Smooth download performance on mobile and desktop.

## 🛠️ Built With
- HTML5 / CSS3
- JavaScript (Vanilla)
- html2canvas for high-res rendering
